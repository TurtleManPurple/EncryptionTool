# 📥 How to Install EncryptionTool on Linux

1. Download [EncryptionTool-x86_64.AppImage](https://github.com/TurtleManPurple/EncryptionTool/raw/refs/heads/main/EncryptionTool-x86_64.AppImage)
2. Run this command BEFORE RUNNIG IT:
```bash
chmod +x EncryptionTool-x86_64.AppImage
```
3. Run the command
```bash
./EncryptionTool-x86_64.AppImage
```
to run it
